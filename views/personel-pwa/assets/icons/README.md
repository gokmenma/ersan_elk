## İkon Oluşturma

1. 512x512 boyutunda bir kaynak ikon oluşturun
2. Köşeleri yuvarlatılmış (maskable) bir tasarım tercih edin
3. Arka plan rengini #135bec yapın
4. Diğer boyutları otomatik oluşturmak için:
   - https://www.pwabuilder.com/imageGenerator
   - https://realfavicongenerator.net/

## Örnek İkon

Mavi arka plan (#135bec) üzerine beyaz "Er" harfi.
